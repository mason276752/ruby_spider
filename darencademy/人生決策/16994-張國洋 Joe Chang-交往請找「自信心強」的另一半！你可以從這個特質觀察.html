我在【戀愛大人學】上課時會聊到一個概念。就是男生通常都有自信心不足的課題。也因此，我通常都會建議女生在選擇交往對象時，請無論如何都請盡量要找自信強的男生。

你說，我怎麼知道男生自信強？他工作強、社會地位高、手上有錢、開好車就是嗎？不一定。社會地位高，有錢，工作好、開名車的很多自信也很不足。

要看男人的自信心，我建議你觀察他最在意的女性特質是什麼。舉例來說，會覺得美麗、漂亮、身材好的，這多半是正常人。意思就是，這個族群的自信心大概就是平均水準上下。因為他們沒想甚麼太複雜的事情，就本能驅動，也沒恐懼害怕甚麼事情（當然也可能純粹是沒想太遠，還沒考慮長期關係，只是想找人交往）。

但如果你碰到一個男生，他覺得「乖巧聽話」是最重要的，那就請務必小心。他要就是有某種自信問題，不然就其實並沒這麼重視你。因為要你聽話，其實就是你怎麼想對他而言並不重要的意思。既然你怎麼想並不重要，那所有的追求與討好都必然只是想你入彀的手段。

我不敢說自己沒有自信問題，我覺得自己多半也有。

但隨著自己一路成長，認知提升，多接觸心理學後，多理解自己的心智狀態後，我發現我對於重視的條件確實有不斷轉換。像我學生時代，因為對自己沒自信、覺得自己很差、擔心女生會拋棄自己，當年真的就也是很想找一個好控制的女朋友 — 她最好能永遠不反駁我，都聽話，很乖順，永遠不會離開我。這想像起來才會讓人覺得安心。

但我後來比較長大了，開始理解長期相處最重要的是「契合度」。但契合度怎麼來？我講座跟課程提的那「關鍵三點」是先決條件外，另一個我主觀覺得很重要的（但未必適用大家就是），是女生要「夠聰明」。

我觀察身邊那些有一定程度自信的男人，他們長期相處的伴侶，一定都是聰明或有智慧的。

可是你或許會疑問，想說聰明的女人不是多半會不聽話？那不就衝突不斷嗎？小時候我也有這樣的疑惑。但隨著我自己自信心慢慢提升，我發現有自信的男人其實真不需要女伴乖巧聽話。因為聰明不聽話、最後卻很契合，這才是「最了不起的契合」。

怎麼說呢？原因大概是這樣。比方說，碰上一個抉擇，有 AB 兩個選擇，你跟你的情人都在猶豫怎麼選。假設你突然說，不然 AB 都不要，選 C 吧。笨的女生就會疑惑，想說「咦？ C 是什麼？哪裡來的 C？為何你選這個？」然後她疑問半天，你詳細解釋了她還都聽不懂；或是勉強聽話但內心滿是不解與委屈。於是，要就是你一直花力氣解釋、比喻、畫圖、講來龍去脈說服她。不然就是你碾壓，要她閉嘴相信你。讓對方無腦聽話雖然很有效率，但互動上其實是很無聊煩悶的，因為一方總是服從，這其實不是高品質的相處。

但聰明的女生就不同。她們可能一聽你提出 C 方案也會疑惑，但因為她們聰明，就會瞬間跟你同步。她們會靈光一閃，在腦中思考完你思考的路徑、很快搞懂你思考的脈絡，然後說：「對耶，C 其實更好，你說的沒錯。」或是她還可以加成 C 的好，會說出：「可是這樣就會有 XX 問題，不如 C 加上 D 會更完美。」甚至她根本就是原始提出 C 方案的那個人，然後你瞬間同步搞懂她的思路。這才是勢均力敵的對等關係。

在這種狀況下，兩人瞬間同步後，你看她、她看你，不用多作解釋、不用說明細節，但彼此瞭然於心，這會很有周瑜跟諸葛亮手心寫火、相識一笑那種互相訝然的情緒。這種與人自然同步的爽快感，你試過一次，你就會發現這比甚麼聽話乖巧來的更加重要！

畢竟，真正聰明有實力的人，都喜歡有人能真心理解自己的聰明並被真心讚賞。你知道，高處不勝寒的，當你可以瞬間判斷某些事情，但周圍的人都是「華生」，非要你從頭開始建構式數學般的解釋清楚時，你會很挫折，也不會享受這種相處的。但周圍人若都只是無腦的聽命，你一樣會覺得很孤獨 — 給你當上大王，但沒人欣賞你的聰明，那這種大王又有甚麼好開心的呢？？所以會想在關係中當大王的，要就根本是笨蛋，要就是對於自己能力其實沒甚麼自信，要就是根本不在意你只是需要小寵物……。

有能力又有自信的，自然會開始在意身邊的人也要是聰明有自信的。而且兩方都是這樣的人，關係的安全感會更提升，相處也會更融洽的。

簡單講：

人生追求碾壓的，多半自信薄弱，也會很多地雷。

人生追求共感的，多半較有自信，也會更易相處。

（註：因為我是男生，以上文章我用男生當主角來說明，但反過來概念也是成立的）
