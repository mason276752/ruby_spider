對多數人來說，取得像是PMP這樣的專案管理認證主要就是希望有更好的職場發展。美國專案管理協會（PMI）發行的期刊（PM Network）中有一篇報導，針對四個國家在專案管理專業的需求及發展作一介紹，這裡摘錄一些內容，讓大家可以和國內的現況作一比較。雖然數據是針對金融風暴前的統計，但我認為還是有相對性的參考價值。

雖然澳幣最近走軟，但近10年來全球原物料價格攀升，占澳洲全國出口額65%的農礦業成為澳洲的經濟火車頭，加上澳洲本身經濟結構相當成熟，服務業占國民生產毛額的68%，所以像是銀行，金融，政府與IT產業都一起大幅成長。對於專案管理專業的需求尤其殷切，尤其是天然資源與營建工程等大型專案，急需要有經驗且具有專業認證的專案管理人員。在專案管理的次領域中，又以變更管理，時程管理與IT專案管理最為熱門。以澳洲國防部為例，他們願意以年薪10萬美金來聘請時程管理人員。而根據PMI的薪資調查，澳洲專案管理工作者年薪的中值將近8萬美元（2008上半年）。

金磚四國的巴西同樣是以原物料出口為主要的經濟引擎，近年來的經濟也相當引人注目。但專案管理的發產起步較晚，對PMP的需求，仍以礦業，金融，製藥，醫療與通訊幾個產業為主，而且集中在比較小型的專案。由於越來越多人開始注重專案管理，協助取得認證的教育單位發展也相當不錯。巴西的職場相當注重人際關係，同事下班後的互動和歐美其他國家相比較頻繁，所以專案工作者處理人際關係的能力變得特別重要。至於薪資方面，中值為6萬8千美金（2008上半年）。

沙烏地阿拉伯是全球最重要的產油國，石油相關的產業當然也是該國最重要的經濟命脈。有關營建工程，煉油，石油氣與化學的大型專案相當多。沙國也相當倚重國外的人才，整個就業市場有550萬人屬於外國工作者。基於以上，專案工作機會可說是相當多的，不過還是以大型跨國公司所釋出為主。由於大型的專案資金龐大且內容複雜，所以特別需要具有PMP認證的專案工作者，尤其在規劃能力，成本估算，風險管理等領域。薪資中值約為6萬美金（2008上半年）。

美國的專案工作可說全世界最多元化的，主要是因為美國經濟體原本就是相當平均地由各種產業支撐。這些產業都對專案管理有不同程度的需求，其中以資訊，銀行，醫療，製藥，金融與汽車產業近來來需求最高。美國是專案管理發展較早的國家，所以專案管理的重點，由以往數量少，但相當大型的專案，逐漸演變成現在，數量繁多，內容各異的小型專案。這些小專案除了個別差異大外，時程往往較短，專案間也相互牽連，而且共享公司資源。正因如此，在專案組合管理，資源管理有經驗的PMP，在職場上就相當吃香。依據PMI的調查，專案工作者的年薪中值為9萬美金（2008上半年）。

這幾個國家我只有美國有親身的經驗。雖然美國整體失業率攀昇，但對專案管理人才還是需求大於供給。我的公司（全美前10大工程顧問，條件不算差的）在紐約地區（800萬人口）要找一個會用Project Controls Specialist（專案控制工程師）找了半年，幾乎沒什麼人來應徵。我幫公司打了幾通電話去挖角，優秀的人也不太願意動。一個來自英國分公司的同事告訴我，在英國做專案管理超過七成是獨立顧問，因為機會多，自己開賺更多。另一個趨勢就如上文所述，美國從以往著重在單一專案的管理，演變成多專案的調控（Portfolio Management）和企業標準流程的建立，也就是所謂企業專案管理（Enterprise Project Management），有這樣經驗的人才又更少了。

另外一項觀察，是美國的公部門，尤其是各州或市級旗下的捷運局，環保局之類，擁有大量公共設施需要維護（地鐵，橋樑，污水廠）。因為長期以來專案執行績效不彰，近年紛紛撥預算進行所謂的專案管理流程改進。這類顧問案常稱做Capital Program Management（基礎設施管理），或是Capital Project Improvement之類的名稱，內容也就是企業專案管理的導入或是專案管理資訊系統（PMIS）的建置。這些案子其實還不小，我聽過的幾個例子都是總金額百萬美金以上的。

個人認為這項專業是大有可為，只要願意耕耘個幾年，累積自己的實力，台灣和其他亞洲國家也會逐漸跟進。
