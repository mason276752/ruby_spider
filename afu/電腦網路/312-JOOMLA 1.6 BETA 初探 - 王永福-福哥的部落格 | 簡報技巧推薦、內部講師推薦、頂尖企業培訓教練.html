Joomla 1.6 Beta 1 於 2010/5/17 釋出了，雖然詳細的功能文件我還沒看到，但我試著在 Localhost 安裝，並大概瀏覽一下它的一些新功能。以下是針對 Joomla 1.6 Beta 的初探分享。(歡迎引用，請註明出處）

1.安裝：安裝的方式跟1.5沒有差很多，很簡單就可以裝好了。

2.前台畫面：Demo的前台畫面越來越簡單了（Simple is good，我喜歡），連顏色也採用黑白配色。右上角提供了將字型放大縮小的功能（按 Reset 復原）。而原來 Read more (閱讀更多）的功能，在文章下方作成像按鈕的樣子（推測可能每一個 template 都不一樣，或是可以自行訂製）。還有，一些預設的廣告Banner & 投票區等功能，沒有顯示在示範網站中（經 check 後台預設元件，Banner 功能還在，但投票功能好像就不見了…）

3.可關閉式選單：在隨意瀏覽 Demo 網站的過程中，我發現了某些選單（如畫面右側 Extensions選單）是可以隨意關閉的，這個功能蠻有趣的。另外在左方 This Site 主選單，也發現了 Site Map 似乎變為預設功能之一

4.這是 Site Map 的文件內容，似乎是提供了一個 Plugin 的方式，來顯示網站地圖

5.後台：後台登入畫面也沒有改變太多

6.後台控制台 ICON 雖然還是10個，但是功能跟以前不太一樣，用分類管理取代了以前的單元＆分類管理（這是 1.6 最大的差異之一）。另外首頁管理也不見了（新版似乎改名叫 Featured articles)

1.5版控制台畫面

7.分類管理：新版的分類管理己不限３級製（單元/分類/文章），現在您想分幾類就分幾類，在畫面中也用縮排的方式來表示

8.文章新增＆編輯：文章編輯畫面跟以前差不多，要注意的是〔發佈首頁〕己經改為〔Featured〕

9.插入圖片：在文章中插入圖片也跟1.5 版相同，用動畫視窗來表示

10.文章權限：文章編輯畫面右側，多了細部的〔Article Permissions〕，可以更仔細的設定文章的操作權限．

11.Featured Articles：這個功能取代了以前的首頁管理

12.選單新增：我試著新增一個選單，操作方法又有點不一樣了。1.5版新增選單時，要先決定選單項目，然後才進入新增畫面（如下圖）

1.6版在新增選單時，進入的就是選單新增畫面．但是在第２項〔Menu Item Type〕，就是原來的選單項目，然後在按右側的 Select Artcile 選擇文章（功能好像有點 Bug，沒有確認鍵）

13.這是選單類別的選擇畫面，功能跟以前又有點不一樣

14.使用者管理：1.6 版將使用者管理獨立成一個選單項目，可能是作了很多權限上的改進跟設定吧。例如可以自建使用者群組

15.包含在全站設定中，也可見細部的權限操作設定（連剛才文章新增時也是）

16.預設元件：這是預設的元件，投票功能不見了（可能是中看不中用吧？），而新出來一個 Redirect 是什麼？還沒研究…

17.模組安裝：模組安裝的選單及功能跟前版似乎差不多，就是單一安裝界面，然後分開管理

18.模組管理：這是模組管理的畫面，跟以前差不多

19.佈景管理：這是佈景管理的畫面，新版將佈景管理又區分為 Style 跟 Template （我是覺得有點多此一舉，還是我不夠了解）。這是 Style 的畫面，用來設定佈景內的參數

20.這是 Templates 的畫面，詳例了目前有那些 Template

以上就是對 Joomla 1.6 Beta 1 的大概瀏覽，由於尚未深入研究（嗯….嗯….可能也不太有時間深入研究了吧），所以只快速的檢視一下比較明顯的差異。目前除了權限設定及不限級數的階層分類，我暫時還沒有發現有什麼很大的不同。也歡迎大家提供建議及指正。
